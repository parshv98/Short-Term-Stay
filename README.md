# Short-Term-Stay

A feature-rich short-term rental platform engineered to deliver a refined booking experience through dynamic listings, optimized search capabilities, and a robust backend system.

## Features

- **Dynamic Listings:** Discover a wide range of short-term rental properties with detailed descriptions, photos, and availability.
- **Optimized Search:** Find stays using powerful filters and search capabilities for dates, amenities, location, and price range.
- **Robust Backend:** Secure and reliable backend for account management, booking system, and rental operations.
- **Modern User Interface:** Responsive design using EJS templating, CSS styling, and interactive JavaScript.
- **Host & Guest Profiles:** Manage listings as a host and bookings as a guest.

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/parshv98/Short-Term-Stay.git
   cd Short-Term-Stay
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **(Optional) Set up database connection**
   - Update configuration files for MongoDB or your preferred database.

4. **Run the application**
   ```bash
   npm start
   ```
   or
   ```bash
   node app.js
   ```

5. **Access locally**
   ```
   http://localhost:3000
   ```
   (Port may vary. Check the configuration.)
